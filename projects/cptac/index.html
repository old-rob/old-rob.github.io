CPTAC Python Package

Payne Lab, BYU | Data Engineering & Architecture

Repository at: https://github.com/PayneLab/cptac

The Challenge

The Clinical Proteomic Tumor Analysis Consortium (CPTAC) generates massive amounts of proteogenomic data. Researchers needed a more reliable, standardized way to access and join these complex datasets.

Key Contributions

  • Dataset Integration: Integrated confirmatory GBM and PDAC cohorts.
  • Core Refactoring: Modernized the codebase, removing deprecated functions and standardizing docstrings for better maintainability.
  • Data Integrity: Implemented 'latest' source logic and fixed critical bugs in join_metadata_to_omics.
  • Domain Logic: Developed specific handling for HNSCC normal tissue types to improve scientific accuracy in annotations.

# Creator of Baffall

**Baffall** is a solo-developed word search game that blends elements of **Tetris**, **Boggle**, and **Bookworm** into a fast-paced, spatial word challenge. Built in **Godot** and published to **Android**, **Itch.io**, and (soon) **iOS**, Baffall delivers an original twist on classic word mechanics through reactive design, fluid interaction, and tight polish.

## Key Contributions

### Game Design and Mechanics

- Designed a unique hybrid gameplay system combining real-time **tile dropping** with spatial **word formation**, creating a balance between speed, vocabulary, and board control.
- Iterated on core mechanics to ensure satisfying play, scaling complexity naturally as the player progresses.

### Programming and Engine Work

- Built the entire game in **Godot**, managing all logic, animations, UI flow, and state handling.
- Implemented custom systems for:
  - **Tile physics and snapping**
  - **Word detection and validation**
  - **Difficulty scaling and game modes**
  - **Player data tracking and leaderboard integration**

### UI/UX and Visual Polish

- Designed and implemented the complete interface, from gameplay to menus and transitions, with a focus on **clarity, responsiveness, and touch-first interactions**.
- Tuned feedback systems — including animations, color shifts, and audio cues — to make each interaction feel intuitive and rewarding.

### Publishing and Distribution

- Published the game to **Android (Google Play)** and **Itch.io**, handling all build optimization, store listings, and platform-specific requirements.
- Currently preparing for **iOS release**, including resolution handling, input tweaks, and platform compliance.

## Impact

**Baffall** showcases my full-stack capability as a solo game developer — from concept and code to design and deployment. It demonstrates a strong command of game systems, user experience, and cross-platform delivery, all built around a compelling core mechanic.

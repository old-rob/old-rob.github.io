Baffall: Word Search Mobile Game

Published: Apr 29, 2026

Project Overview

Baffall is a solo-developed word search game that blends elements of Tetris, Boggle, and Bookworm into a fast-paced, spatial word challenge. Built in Godot and published to Android, Itch.io, and iOS, Baffall delivers an original twist on classic word mechanics through reactive design, fluid interaction, and tight polish.


Key Contributions

  • Designed a unique hybrid gameplay system combining real-time tile dropping with spatial word formation.
  • Implemented custom systems for tile physics, word detection, and difficulty scaling in Godot.
  • Published the game to Android (Google Play) and Itch.io, integrating AdMob for advertising.

Impact

Baffall showcases my full-stack capability as a solo game developer — from concept and code to design and deployment. It demonstrates my ability to take an idea and create something real with my understanding of game systems, user experience, and cross-platform delivery.
